Aide :
	But du jeu:
		Avoir le plus de pays suivants votre cause.
	Plateau de jeu:
		Le plateau se compose de 20 cases positionnées en carré.
		Il est composé de 3 types de cases:
			- les pays : vous pouvez ici effectuer des actions pour augmenter votre nombre de partisans dans la ville.
			- les chances : une action aléatoire se produit, bonne ou mauvaise.
			- les impots : cette case est unique, elle vous fait perdre 50% de vos dollarzz.
